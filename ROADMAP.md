# Roadmap

AI Agent Studio 的目标是从课程资料仓库，逐步演进为可公开访问、可自学、可演示、可评测的 Agent 开发实战工作台。

## Phase 1: Public Launch

- GitHub README 具备清晰课程定位、路线图和能力矩阵。
- `docs/` 提供可部署的公开官网。
- `docs/studio/` 提供静态版 AI Agent Studio。
- `docs/showcase/` 提供 L12 智能客服 Agent 项目展示页。
- README、Launch Kit 和公开截图支持对外传播。
- CI 检查课程索引、公开站点、敏感信息和静态资源。
- 每章保留稳定结构：讲义、材料、实战、资源和交付物。

## Phase 2: Learning Loop

- 已为 Studio 增加学习报告导出。
- 已增强小测和面试题抽卡体验。
- 已增加章节完成度和 L12 毕业项目展示面板。
- 为每章加入更明确的“课前 / 课中 / 课后”路径。

## Phase 3: Evaluation And Project Review

- 将 L11 评测能力接入课程平台。
- 为 L12 智能客服 Agent 增加一键评测结果展示。
- 增加项目提交规范、评分 Rubric 和优秀答案样例。
- 建立 bad case 收集和课程内容回流机制。

## Phase 4: Public Launch Plus

- 支持云端部署和自定义域名。
- 引入可选用户系统和云端学习进度。
- 提供企业内训版学习包和项目陪跑材料。
- 将课程 RAG 助手升级为可配置知识库问答服务。

## Non-Goals

- 不把浏览器端变成任意 shell 入口。
- 不在仓库内保存真实 API key、代理配置或个人路径。
- 不为了炫技引入重型前端框架，除非功能需求已经超过当前静态架构。
